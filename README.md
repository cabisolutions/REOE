## Empezando

### Requisitos previos

Antes de ejecutar el proyecto necesitas tener instalado y funcionando:
* php
* composer
* apache2
* mariadb

### Instalación

1. Clona el repositorio 
   ```sh
   git clone https://github.com/cabisolutions/REOE.git
   ```
2. Instalar librerias de composer
   ```sh
   composer install
   ```
